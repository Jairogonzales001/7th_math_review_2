Real Numbers
Rational, Irrational & Integers

Quick Reference

Rational Numbers

Can be written as a fraction a/b where a and b are integers and b ≠ 0

  • Decimals that terminate: 0.5, 0.25, 3.75
  • Decimals that repeat: 0.333..., 0.1666...
  • All integers, fractions, and mixed numbers

Irrational Numbers

CANNOT be written as a fraction - decimals go on forever without repeating

  • π = 3.14159265358979...
  • √2 = 1.41421356...
  • √3, √5, √7 (non-perfect square roots)
Key Points:
  • √n is rational only if n is a perfect square (1, 4, 9, 16, 25...)
  • Rational + Irrational = Irrational
  • Rational × Irrational = Irrational (unless rational = 0)

Laws of Exponents
Integer & Variable Exponents

Quick Reference

Product Rule
xa · xb = xa+b
Quotient Rule
xa ÷ xb = xa-b
Power Rule
(xa)b = xa·b
Zero Exponent
x0 = 1 (x ≠ 0)
Negative Exponent
x-a = 1/xa
Product to Power
(xy)a = xaya
Remember: When multiplying same bases, ADD exponents. When dividing, SUBTRACT. When raising a power to a power, MULTIPLY.

Scientific Notation & Radicals
Operations with Scientific Notation & Order of Operations with Radicals

Quick Reference

Scientific Notation

Form: a × 10n where 1 ≤ |a| < 10

  • Large numbers: Positive exponent (3,400,000 = 3.4 × 106)
  • Small numbers: Negative exponent (0.00056 = 5.6 × 10-4)
  • Multiply: Multiply coefficients, add exponents
  • Divide: Divide coefficients, subtract exponents

Order of Operations with Radicals

PEMDAS still applies - treat radicals like parentheses

  • Simplify inside radical first
  • √(a·b) = √a · √b
  • √(a/b) = √a / √b
  • √a2 = |a|
Common Perfect Squares: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100, 121, 144

Equivalent Expressions
Factoring & Multiplying Linear Expressions

Quick Reference

Factoring Common Monomials

Find the GCF of all terms and factor it out

  • 6x + 9 = 3(2x + 3)
  • 4x2 - 8x = 4x(x - 2)
  • 15xy + 10y = 5y(3x + 2)

Multiplying Linear Expressions

Use the distributive property (FOIL for binomials)

  • a(b + c) = ab + ac
  • (x + 2)(x + 3) = x2 + 5x + 6
  • Combine like terms at the end
FOIL: First, Outer, Inner, Last - then combine like terms

Equations & Inequalities
Linear Equations, Inequalities, Quadratic & Cubic Equations

Quick Reference

Multi-Step Linear Equations

  • One Solution: Variable equals one value (x = 5)
  • No Solution: Variables cancel, false statement (0 = 3)
  • Infinite Solutions: Variables cancel, true statement (0 = 0)

Two-Step Inequalities

  • Solve like equations, but...
  • FLIP the sign when multiplying or dividing by a negative!
  • < means "less than" (open circle)
  • ≤ means "less than or equal" (closed circle)

Quadratic & Cubic Equations

x2 = p: x = ±√p (remember both solutions!)
x3 = q: x = 3√q (only one real solution)
